怎么用（最短路径）
  1. 在任意 App 复制一段文字（聊天记录/网页正文都行）。
  2. 点输入框唤起 IME，点击剪贴板提示，进入「剪贴板动作」菜单。
  3. 选择 Bridge Debugger · Inspect Clipboard。
  4. 回到本页面：应收到 binding.invoke，且下方「测试文本」会自动填入 clipboardText。

说明：这是调试功能件，用来验证宿主桥协议/权限/数据形状是否正确，不是业务功能件。
